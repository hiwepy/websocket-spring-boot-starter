# websocket-spring-boot-starter


### 说明


 > 基于spring-websocket 的 Spring Boot Starter 实现

1. 默认的Handler实现
2. 实现基于责任链式的消息分发


### Maven

``` xml
<dependency>
	<groupId>com.github.hiwepy</groupId>
	<artifactId>websocket-spring-boot-starter</artifactId>
	<version>2.0.0.RELEASE</version>
</dependency>
```
